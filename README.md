# mOT Homebrew tap

Install [mOT](https://getmot.app) — the macOS Opinionated Terminal — via Homebrew.

```sh
brew install --cask marcusmathiassen/tap/mot
```

Or tap once, then use the short name:

```sh
brew tap marcusmathiassen/tap
brew install --cask mot
```

Upgrade (mOT also self-updates via Sparkle, so this is rarely needed):

```sh
brew upgrade --cask mot
```

Uninstall (add `--zap` to also remove `~/.mot.yaml` and caches):

```sh
brew uninstall --cask mot
```

## For maintainers

`Casks/mot.rb` is generated by `make cask` in the
[main repo](https://github.com/MarcusMathiassen/mot) — it stamps the latest
`APP_VERSION` and the DMG's sha256 into `mot.rb.in`. After each `make release`,
copy the regenerated `Casks/mot.rb` here and push.
